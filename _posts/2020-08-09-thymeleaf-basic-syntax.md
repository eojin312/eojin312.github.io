## th:text
엘리먼트 사이에 값을 할당한다. jquery 에 text() 와 유사

## th:if, th:unless
말 그대로 if else 역할이고 조건에 해당하는 블럭만 노출된다.

## th:value
input 이나 textarea 의 value 에 할당한다. jquery 의 val() 과 유사

## th:utext
엘리먼트 사이에 html 형태의 코드를 할당. jquery 의 html() 과 유사

## th:with
변수를 재정의

## th:switch, th:case
switch, case 문

## th:fagment
include 역할

## th:attr
attribute 를 thymeleaf 로 정의할 때
